Это пример поста, чтобы проверить вёрстку блога. Здесь можно писать обычный текст,
абзацы, списки.

## Пример таблицы

| Раздел  | Назначение          |
| ------- | -------------------- |
| Блог    | Короткие заметки      |
| Рассказы | Законченные истории  |
| About   | Информация об авторе  |

## Пример картинки

Обложка поста указывается в frontmatter (`image:`) и выводится автоматически.
Картинки внутри текста вставляются обычным Markdown-синтаксисом:
